# TA Technical Assessment (Levels 1-4)

## Remote Format

- The assessment is conducted remotely over Zoom/Meet.
- The evaluator shares their screen for the full session.
- The evaluator does all keyboard and mouse actions and writes all code changes.
- The candidate does not type code directly.
- The candidate gives step-by-step guidance to the evaluator, acting as the TA.
- The candidate should explain reasoning, debugging decisions, and tradeoffs while coaching.

## Candidate Instructions

- Complete problems in order from Level 1 to Level 4.
- Instruct the evaluator on how to fix the code in each file.
- Keep solutions readable for beginner students.
- Do not remove requirements from the prompt text.

## Problem Order by Level

- Level 1: `problem1.html`, `problem2.html`
- Level 2: `problem3.html`, `problem4.html`
- Level 3: `problem5.html`, `problem6.html`
- Level 4: `problem7.html`, `problem8.html`

## What We Evaluate

- Technical correctness.
- Debugging process and edge-case handling.
- Clarity of explanations written for students.
- Code style appropriate for instruction.

## Time Limits

- Level 1: 10 minutes
- Level 2: 15 minutes
- Level 3: 20 minutes
- Level 4: 25 minutes
